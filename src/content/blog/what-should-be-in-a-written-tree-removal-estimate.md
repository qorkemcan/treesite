# What Should Be in a Written Tree Removal Estimate?

A useful written tree removal estimate does more than name a price. It identifies the tree, defines the work, explains what will be removed from the property, and makes clear who is responsible for permits, access, cleanup, stump grinding, and changes in scope.

For most Florida homeowners, the written document should answer ten practical questions before anyone starts cutting.

This page is the written-document checklist. For a broader quote-comparison guide, read [What Should Be Included in a Tree Removal Estimate in Florida?](/blog/what-should-be-included-in-a-tree-removal-estimate-in-florida/).

## The 10 items a written estimate should cover

1. **Who is doing the work?** The business name should match the invoice, payment instructions, and insurance documents.
2. **Which tree is included?** Identify each tree clearly.
3. **What is the work scope?** Removal, hazard reduction, pruning, debris only, or stump work.
4. **How will the tree be handled?** Felling, climbing, rigging, bucket truck, crane, or hand-carrying.
5. **What cleanup is included?** Branches, palm fronds, logs, chips, trunk sections, sawdust, raking, and driveway cleanup.
6. **Is stump grinding included?** [Stump grinding services](/services/stump-grinding/) should be included, excluded, or optional.
7. **Who handles permits or HOA approval?** Do not assume the crew is handling this unless the document says so.
8. **What can change the price?** Hidden decay, access problems, utilities, storm tension, extra hauling, or added work.
9. **What are the payment terms?** Deposit, balance, methods, cancellation rules, and due dates.
10. **What is the final site condition?** The estimate should explain what the property will look like when the crew leaves.
